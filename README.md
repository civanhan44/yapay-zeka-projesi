# yapay-zeka-projesi
# 🩺 Hastalık Tahmin ve Bilgilendirme Sistemi

Bu proje, kullanıcıdan alınan belirtilere göre olası hastalıkları tahmin eden ve hastalıklar hakkında bilgilendirme sağlayan bir uygulamadır. Amaç, kullanıcıların sağlık durumları hakkında ön bilgi edinmesini sağlamaktır.

---

## 🚀 Özellikler

- 🔍 Belirtiye göre hastalık tahmini  
- 📋 Hastalık hakkında bilgi gösterimi  
- 💡 Belirti - hastalık eşleştirme sistemi  
- ⚡ Hızlı ve kullanıcı dostu arayüz  
- 🧠 Geliştirilebilir yapı  

---

## 🛠️ Kullanılan Teknolojiler

- PHP  
- HTML / CSS  
- JSON  
- JavaScript (opsiyonel)
